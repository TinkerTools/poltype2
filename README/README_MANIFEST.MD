### Poltype Working Files

| File pattern | Description |
|-------------|-------------|
| `.sdf` | Input structure file |
| `-opt_1.psi4` / `-opt_1.com` | QM input for first geometry optimisation (Psi4 / Gaussian) |
| `-opt_1.log` | QM output for first geometry optimisation |
| `-opt_1.chk` / `-opt_1.fchk` | Electron density (binary / human-readable) |
| `-opt_2.*` | Second geometry optimisation (same pattern) |
| `-dma.*` | DMA single-point calculation files |
| `-esp.*` | ESP single-point calculation files |
| `.gdmain` | GDMA input |
| `dma.punch` | GDMA intermediate output |
| `.gdmaout` | GDMA output |
| `.key` | Initial key file with multipoles and polarise terms |
| `.grid` | ESP grid coordinates |
| `.cube` | QM potential at grid points |
| `.pot` | Tinker-readable potential file |
| `-peditin.txt` | POLEDIT frame input |
| `RMSPD.txt` | QM vs MM potential RMSD |
| `-groups.txt` | Input for multipole averaging |
| `.xyz_2` / `.key_2` | Structure and key file with symmetry types |
| `.key_3` | Key file after ESP fitting |
| `.key_4` | Key file with valence parameters from database |
| `qm-torsion/` | QM and MM torsion scan data |
| `.key_5` | Key file with fitted torsion parameters |
| `MMDipole.txt` | MM dipole information |
| `QMDipole.txt` | QM dipole information |
| `final.xyz` | Final QM-optimised structure |
| `final.key` | Final key file with all parameters |
| `final.xyz_2` | Final MM-minimised structure |
| `*-poltype.log` | Poltype workflow log |

### qm-torsion

Middle-bond atoms `b_c` identify each torsion scan set. `angle` is the
dihedral angle being scanned.

| File pattern | Description |
|-------------|-------------|
| `-opt-b_c_*_angle_preQMOPTprefit.xyz` | Tinker XYZ before QM opt and fitting |
| `-opt-b_c_*_angle_preQMOPTprefit.key` | Key with torsion restraints (pre-fit) |
| `-opt_1-b_c_*_angle.*` | First torsion geometry optimisation |
| `-opt_2-b_c_*_angle.*` | Second torsion geometry optimisation |
| `-opt-b_c_*_angle_postQMOPTprefit.*` | After QM opt, before fitting |
| `-sp-b_c_*_angle.*` | Single-point calculation files |
| `-opt-b_c_*_angle_postQMOPTpostfit.*` | After QM opt and fitting |
| `-fit-b-c.png` | Fourier fit plot (QMâˆ’MM vs dihedral) |
| `-energy-b-c.png` | Energy comparison plot |

| `parentsymclasstofragsymclass.txt` | Parent-to-fragment type class map |
| `classkeytotorsionindexes.txt` | Type key to torsion atom indices |

### vdw-refinement

| File pattern | Description |

| `classkeytoatomindexes.txt` | Type key to atom index in SMARTS |
| `missingvdw.txt` | Atom indices with missing vdW parameters |
| `-water_a_b.xyz` | Water-molecule dimer structure |
| `-water_a_b.com` / `.psi4` | Dimer QM optimisation input |
| `-water_a_b.log` | Dimer QM output |
| `-water_a_b_X.xyz` | Shifted probe at distance fraction X |
| `-water_a_b_X_sp.*` | Single-point files for dimer |
| `QM_DATA` | QM energies for each probe distance |
| `SP.dat` | Tinker energies for each probe distance |
| `INITIAL.PRM` | Initial vdW parameters and fitting bounds |
| `EnergyVsDistance-*.png` | Energy vs distance plot |
| `QMvsAMOEBA-*.png` | QM vs MM energy comparison |
